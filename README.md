## 🧑‍💻 自己紹介

2014年からJavaを中心としたバックエンド開発に従事しています。

これまでに、業務システム、ECサイト（オンラインショッピングサイト）、SaaS型サービスなど、さまざまなWebアプリケーションの開発に携わってきました。

主にSpring BootやMyBatisを用いたバックエンド開発を担当し、API設計、データベース設計、認証・認可処理などを含むサーバーサイド開発を経験しています。

現在は日本の企業にて業務システムの開発に携わり、既存システムの機能改善、リファクタリング、パフォーマンス最適化などにも取り組んでいます。

また、個人プロジェクトとしてSpring Bootベースのブログシステム開発を行っており、RBAC権限管理やJWT認証に加え、DDD（Domain Driven Design）を意識したアーキテクチャ設計にも取り組んでいます。

# 🚀 Main Personal Project

## 🚀 manpower-blog

Spring Boot + Vue3 によるブログシステム（企業レベル設計を意識した個人開発）

### 🧩 Architecture

- DDD（Domain Driven Design）を意識したマルチモジュール構成
- system / content / infra で責務分離
- RESTful API設計（統一レスポンス構造）
- グローバル例外ハンドリング

### 🔐 Security

- Spring Security + JWT による認証・認可
- RBAC（ユーザー・ロール・メニュー）権限管理
- 動的メニュー制御（フロント連携）

### ⚙️ Technical Highlights

- MyBatis Plus による効率的なデータアクセス設計
- Redis を利用したキャッシュ・セッション補助
- AES-256-GCM による機密データ暗号化
- MDC（traceId / operator）によるログ設計
- 設定値の暗号化（Jasypt対応可能）

### 🖥️ Frontend

- Vue 3 + TypeScript + Element Plus
- 動的ルーティング + 権限制御
- Pinia による状態管理

### 📌 Features（※補助的に少しだけ）

- ユーザー管理
- 記事・コメント・カテゴリ管理
